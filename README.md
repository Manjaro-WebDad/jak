## Getting Started

* Prerequisites
* Python  >= 3.6
* PySide2 >= 5.12.3
* QTWebEngine

Installing
```
git clone this_repo_address
cd this_repo_directory_on_your_local_machine
pip install -r requirements.txt
~/.virtualenv/python setup.py install
```
or
```
sudo setup.py install
```
